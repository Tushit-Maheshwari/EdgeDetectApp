# EdgeDetectApp

Real-time Canny edge detection on Android using OpenCV, JNI, and OpenGL ES.

## Quick Setup

1. Update OpenCV path in `app/src/main/cpp/CMakeLists.txt`
2. Open in Android Studio
3. Build and run

## Features

- Real-time camera processing
- Toggle between raw and edge modes
- Hardware-accelerated rendering
- FPS counter
- TypeScript web viewer

## Web Viewer

cd web && npm install && npm run build && npm run serve
